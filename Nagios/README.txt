Le Script InstallAuto.sh est un script qui permet d'automatiser l'installation de de Nagios 3.4.1, Nagios plugins 1.4.16 et de NRPE 2.13 sous Debian 6.

Comment utiliser ce script?

dans un terminal tapez:

wget https://raw.github.com/tubezleb/scripts/master/Nagios/AutoNagios.sh
chmod +x AutoNagios.sh
./AutoNagios.sh
